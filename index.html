HTML

  • Data in the body element is what shows on the webpage
  • The head element contains metadata about the webpage
  • The title of the webpage is the visible element of the head element
  • Metadata is not visible to the user
  • We use semantic HTML elements to indicate the role of their content

CSS

  • CSS stands for cascading style sheets
  • CSS styles your webpage so that is it visually appealing
  • the id attribute of HTML elements can be used to style specific sections of the webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system used for web development projects
  • Versions are tracked in GitHub, a cloud repository
  • On Windows, Git can be accessed with Git Bash commands
  • Branches, folders, and files can be created through Git

JavaScript

  • Javascript coding helps a user interact with a webpage
  • Javascript can be included within HTML code, but it's best practice to create a separate .js file
  • Javascript is the most widely used language for front end web development
  • Javascript functions and objects are complex topics.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
